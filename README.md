# Lab work for EE147: GPU Programming and Computing

Each of the assignments provided me the code except the kernel. We were required to write the kernel from scratch. Each assignment focused on a specific techniques within GPU programming. This course provided the basics of understanding the advantages of GPUs as well as demonstrating the hardware aspect in a concise but informative manner.

## Assignment 1
- Thread Indexing
- Host-device memory transfer
- Kernel Launch Configuration
## Assignment 2
- Shared memory
- thread synchronization ( __synchthreads() )
- Parallel reduction pattern
- Block-level partial sums
## Assignment 3
- 2D grid and block indexing
- Memory access pattern
- Global memory reads
- Tiling/shared memory
## Assignment 4
- Race condition prevention (atomicAdd)
- Global memory writes
- Data-dependent memory access
